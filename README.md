# Stargate Datacenter Build Timelapse

Interactive visualization tracking the construction of the Stargate Datacenter through satellite imagery.

**[View Live →](https://calebparikh.xyz/stargate-timeline/)**

## Overview

Satellite timelapse of the Stargate Datacenter construction from August 2024 to February 2026.

- **Location:** 32.5022°N, 99.7892°W (Texas)
- **Imagery:** Sentinel-2 via Copernicus Data Space
- **Timeline:** August 2, 2024 – February 2, 2026

## Features

- Interactive timeline slider
- Playback controls
- Scale reference (120-yard football field)
- Downloadable GIF export

## Development

Built with [Astro](https://astro.build).

```sh
npm install
npm run dev        # Start dev server at localhost:4321
npm run build      # Build for production
npm run preview    # Preview production build
```
